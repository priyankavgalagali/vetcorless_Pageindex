# Reference

This page is included to test tracking across multiple Markdown routes.

## Sample sections

### Analytics
All pages using `layout: default` will load the same analytics code.

### Unique users
The visible count panel is shared across pages because it is rendered by the layout.

### Routing
Each page has its own URL, so page-level visits can still be differentiated.
